Adicionando imagens

<img> Image / Imagem

Em termos de tradução, "img" é uma abreviação de "image" em inglês, que pode ser traduzido para "imagem" em português.
A tag HTML "img" é usada para exibir imagens em uma página da web, a mesma não possui uma tag de fechamento e é uma tag vazia, ou seja, ela não tem conteúdo entre as tags de abertura e fechamento.

src: Source / Fonte / Origem

A tradução do termo, "src" é uma abreviação de "source" para o português é "fonte" ou "origem".
Por exemplo, no caso da tag "img"; o atributo "src" é usado para especificar o caminho e o nome do arquivo de imagem que deve ser exibido na página.

widht e height: Largura e Altura

A tradução dos termos "width" e "height" para o português é "largura" e "altura", respectivamente.
O atributo width é usado para especificar a largura da imagem em pixels, e o atributo height é usado para especificar a altura da imagem em pixels.
É recomendável sempre especificar as dimensões da imagem usando ambos os atributos width e height, para garantir que a imagem seja exibida corretamente e para melhorar a performance da página.
