Aula 11: Criando Variáveis

Introdução

Agora que mergulhamos em como estruturar e estilizar páginas da Web, vamos dar uma olhada no que precisamos para torná-las interativas.

Analisaremos detalhadamente o Javascript, uma linguagem de programação que alimenta sites e aplicativos dinâmicos.

Sites dinâmicos precisam se lembrar de informações para exibir ou alterar. Para isso, o JavaScript possui variáveis.
Como as caixas móveis, as variáveis ​​têm conteúdo e nomes que nos dizem o que há dentro.

const e let

Usamos let e const para criar variáveis. Usaremos ambos por enquanto e cobriremos a diferença entre eles mais tarde.

Toda variável precisa de um nome.
Os nomes das variáveis ​​precisam ser palavras únicas e, portanto, sem espaços.

Para criar nomes de variáveis ​​com várias palavras, usamos camelCase. Começamos com uma letra minúscula e colocamos em maiúscula as palavras adicionais.

Exemplo:
let homeCity

Para nos ajudar a entender o que há dentro de uma variável, escolhemos nomes descritivos.

Questões

Por que damos às variáveis ​​nomes descritivos como city ou country em vez de x ou y?
Para nos ajudar a entender o que há dentro deles.

Que palavras podemos usar para declarar variáveis?
Let e Const.

Para que usamos a Camel Case?
Para criar nomes de variáveis com várias palavras.

Continuação

Os nomes das variáveis ​​também podem conter números. A adição de números é útil com várias variáveis ​​semelhantes.
Depois de criar e nomear uma variável, usamos o sinal = para armazenar um valor dentro dela.
Para terminar de criar uma variável, colocamos um ponto e vírgula ";" no final da linha.

Exemplo:
const city ="Chicago";

Os valores que estamos armazenando "Chicago" são strings. Strings são palavras entre aspas duplas.
Strings podem conter todos os tipos de letras e símbolos, incluindo espaços.

Questões

Como sabemos que um valor é uma string?
É uma palavra entre aspas duplas.

Diferença entre const e let

Já sabemos que podemos declarar variáveis ​​com const e let. Vamos descobrir a diferença entre eles!
Usamos const para declarar variáveis ​​cujos valores não devem mudar.
const é a abreviação de "constante".
Se tentarmos atualizar um valor armazenado em uma variável const. Receberemos um erro.

No entanto, podemos atualizar variáveis let ​​com o = operador de atribuição.

Para deixar claro que uma variável é do tipo const e não deve mudar, podemos opcionalmente nomeá-la com letras MAIÚSCULAS.

Não podemos usar camelCase se usarmos esse padrão de maiúsculas, então usamos maiúsculas snake_case para nomear variáveis ​​com mais de uma palavra.

Exemplo MAIÚSCULA:
const BIRTHDAY = "24/02/1993";

Exemplo SNACK_CASE:
const SPEED_OF_LIGHT = "3*10⁸ m/s";

Ambos const e let são variáveis ​​úteis e poderíamos usá-los de forma intercambiável. No entanto, devemos tentar usar const como padrão.

Linhas de Códigos

Linhas de código são instruções para o computador seguir.
Quando executamos o código, dizemos ao computador para seguir as instruções que montamos.
Com a instrução especial console.log(), dizemos ao computador para exibir um valor em uma área chamada console.

Exemplo:
console.log ("Hello, World!")

Podemos usar a instrução console.log() quantas vezes quisermos. O computador exibe todos os valores em uma linha no console.
Podemos usar console.log() para exibir variáveis ​​como greeting, também.
Quando exibimos variáveis ​​no console, seus valores aparecem em vez de seus nomes.
