Definindo Tamanho e Bordas

Quando bem feitos, os sites fornecem as informações certas de maneira rápida e fácil.
Vamos aprender sobre uma das maneiras mais fáceis de priorizar informações: alterar o tamanho dos elementos ou adicionar bordas.
Para alterar a altura de um elemento, usamos a propriedade height.
Para alterar a largura de um elemento, usamos a propriedade width.
Medimos a altura e a largura de um elemento em pixels, como 50px. Pixels são pequenos pontos que compõem o que você vê na tela.
Podemos alterar a altura e a largura de qualquer elemento em uma página da web.


Qual é a unidade de medida para o tamanho dos elementos?
Pixeis.

O que significa 50 pixels?
50px.

Qual propriedade altera a altura de um elemento?
Height.

O que acontece quando definimos height: 100px?
Todos os elementos tem sua propriedade height definida como 100px.


Os elementos em uma página da Web podem ter bordas ao redor deles.
Para ver como fica, adicione a propriedade border.
Para fazer a borda aparecer, definimos o valor como solid.
Podemos definir a largura de uma borda adicionando um número em pixels logo após solid.
Podemos tornar essa borda vermelha definindo o valor da cor como red.
border-radius é uma propriedade que arredonda os cantos de um elemento.
border-radius funciona em todos os elementos, até mesmo imagens. É uma maneira fácil de fazer com que as imagens fiquem ótimas em uma página da web.
Para tornar uma imagem um círculo, definimos border-radius a metade da largura de uma imagem. Só funciona se a imagem for um quadrado!

Exemplo:
height: 100px;
width: 100px;
border-radius: 50px;
border: solid 5px;

No exemplo acima teremos uma imagem quadrada se tornado um circulo com uso do border-radius com valor da metade da largua da imagem.


O que é uma borda em CSS?
Uma linha de uma certa espessura e a cor em torno de um elemento

Para fazer aparecer a borda, qual valor é necessário?
O valor solid é necessário.

Como arredondamos os cantos de um elemento?
border-radius.

Como definimos a cor de uma borda?
Adicionando um valor de cor a border.
