Estruturação de texto com tags


<br> Line break / Quebra de Linha

A tag "br" é uma abreviação para "break", que significa "quebra" ou "interrupção" em inglês.
Às vezes queremos formatar parágrafos com quebras de linha.
Podemos separar linhas com a ajuda da tag de quebra de linha: <br>.

Exemplo:

Este texto foi
quebrado com a tag "br",
enfatizado com a tag "em",
se tornou importante com a tag "strong"
tudo isso feito com tag HTML

"br" é uma tag vazia, tags vazias não possuem tag de fechamento e nenhum conteúdo.
As tags vazias em HTML são:

  • <area>
  • <base>
  • <br>
  • <col>
  • <embed>
  • <hr>
  • <img>
  • <input>
  • <link>
  • <meta>
  • <param>
  • <source>
  • <track>
  • <wbr>
Eles também são chamados de tags de fechamento automático.

<em> Emphasis / Ênfase / Italíco

Existe um elemento para dar ênfase ao texto e torna-lo itálico. É chamado de "em".
A tag "em" é uma abreviação para "emphasis", que significa "ênfase" em inglês.
"em" é utilizada para enfatizar um texto em um documento HTML,
indicando que aquele trecho de texto deve ser lido com mais ênfase ou importância

<strong> Strong / Negrito

Para definir textos como importantes, usamos as tags "strong" para torna o texto em negrito.
A tag "strong" significa "forte" ou "importante" em inglês.

<head> Head / Cabeça

O "head", é um contêiner de informações sobre uma página da Web, como seu título. Vai antes da tag do "body".

Exemplo:


A tag "head" é usada para incluir várias informações importantes sobre a página, como:
título da página <title>,
codificação de caracteres <meta charset>,
descrição da página <meta name="description">
link para um arquivo de estilo externo <link>.
Essas informações são importantes para ajudar os motores de busca a indexar a página corretamente, além de serem úteis para os usuários que utilizam leitores de tela.

<title> Title / Título

A tag "title" é usada para definir o título de um documento, que é exibido na aba do navegador ou na janela do navegador quando a página é exibida.
Em termos de tradução, "title" significa "título" em inglês.
É importante lembrar que o texto incluído na tag "title" deve ser conciso e descritivo, ajudando os usuários a entenderem o conteúdo da página de forma rápida e clara.
Além disso, o texto incluído na tag "title" também é usado pelos motores de busca para indexar a página, portanto, é importante incluir palavras-chave relevantes para a página no título.

<!doctype html>

O doctype informa ao navegador da Web qual versão do HTML estamos usando. Sem ele, o navegador pode não exibir a página corretamente.
"doctype" significa "tipo de documento" em inglês
Na verdade, a tag <!DOCTYPE html> não é considerada uma tag HTML em si, mas sim uma declaração que informa ao navegador que o documento está usando a versão mais recente do HTML, que é o HTML5. A declaração do tipo de documento (doctype) é necessária para garantir que o navegador interprete corretamente o código HTML do documento.
