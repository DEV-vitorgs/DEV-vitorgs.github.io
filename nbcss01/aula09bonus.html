A qual propriedade o valor italic pertence?
font-style.

Como alteramos o tamanho do texto em uma página da Web?
font-size.

Por que usamos etiquetas style?
Eles nos permitem estilizar vários elementos ao mesmo tempo.

O que é uma regra CSS?
Um seletor e suas declarações.

Qual seletor usamos para selecionar todos os h1 elementos em uma página da web?
O h1 seletor de tags.

O que acontece com uma página da Web se não incluirmos um link para um determinado arquivo CSS?
Os estilos no arquivo não se aplicam á página da web.

Para adicionar uma borda ao redor de um elemento, qual valor é necessário?
O valor solid.
