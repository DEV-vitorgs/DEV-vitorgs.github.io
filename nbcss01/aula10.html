Adicionando Padding

Qualquer elemento tem quatro preenchimentos contados no sentido horário de cima para a direita, para baixo e para a esquerda.
Até agora, aprendemos a definir o preenchimento para cada lado de um elemento, um de cada vez.
Podemos definir cada valor de preenchimento em uma linha usando a propriedade padding e um valor de pixel para cada lado.

Exemplo:
padding: 20px 20px 20px 20px;

Para posicionar o conteúdo mais baixo dentro de um elemento, adicione um preenchimento superior definindo o primeiro valor como 100px.
Para definir o preenchimento a direita alteramos o segundo valor.
Para definir o preenchimento a esquerda alteramos o quarto valor.
Para definir o preenchimento da parte inferior alteramos o terceiro valor.
Quando não queremos adicionar preenchimento a um lado, podemos adicionar 0 e deixar de lado px, pois é opcional.

Como definimos valores para padding?
No sentido horário, cima, direita, baixo e esquerda.

O que acontece quando definimos um padding 0 e deixarmos de fora o px?
px é opcional quando o valor é 0.

Border-radius e margin

As abreviações funcionam com as propriedades margin e border-radius, possibilitando a criação de estilos interessantes com menos linhas de código.
Para definir as margens direita e esquerda 15px em uma linha, podemos usar margin; da mesma forma que fazemos com padding:
margin: 0 15px 0 15px;

Para adicionar as margens superior e inferior, definimos o primeiro e o terceiro.

margin: 40px 15px 40px 15px;

A propriedade border-radius com quatro valores arredonda os cantos no sentido horário a partir do canto superior esquerdo.
Podemos arredondar apenas o canto superior esquerdo alterando o primeiro valor.
Para levar border-radius ao extremo, podemos definir um canto para o mesmo valor que a altura ou largura.
Podemos usar border-radius para dar um toque único às imagens fazendo coisas como arredondar os cantos superior direito e inferior esquerdo.

Qual é a ordem dos valores para a margin?
No sentido horário, cima, direita, baixo e esquerda.

Qual é a ordem de valores para a propriedade border-radius?
Superior esquerdo, Superior direito, Inferior direito, Inferior esquerdo.
