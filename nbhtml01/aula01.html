Descobrindo HTML e Tags


O código HTML informa aos navegadores da web como exibir conteúdo como imagens, links e texto em uma página da web. HTML é uma linguagem de marcação de computador que estrutura todas as páginas da Internet.


Já aprendemos o suficiente para responder à nossa primeira pergunta do questionário! Qual linguagem de marcação estrutura todas as páginas da web na internet?
Resposta: HTML.


Em HTML, as coisas giram em torno de tags. As tags começam com <, seguidas por uma palavra-chave e terminam com >.
Qual tag HTML cria um botão?
Resposta: </button>


<button> é uma tag de abertura. Ele informa ao navegador da Web que queremos começar a exibir um botão.
A maioria das tags em HTML vem em um par, elas têm uma tag de abertura e uma tag de fechamento.
O que você codificou anteriormente é a tag de abertura de um elemento de botão. Vamos codificar a tag de fechamento do botão adicionando "/".
Exemplo: </button>


Em HTML, as palavras-chave de algumas tags são óbvias, mas existem algumas tags que usam uma forma abreviada.
Por exemplo, usamos <p> para um parágrafo.
Qual tag HTML cria um parágrafo?
Resposta: <p>


Codificar um parágrafo é muito semelhante a codificar um botão. Você precisa de uma tag de abertura, seguida do seu texto e uma tag de fechamento.


Podemos criar cabeçalhos em HTML para enfatizar determinados textos. Existem 6 tamanhos de cabeçalho em HTML, de h1 a h6.
Qual tag HTML cria um cabeçalho?
Resposta: <h1>


A tag de título enfatiza os textos, tornando-os maiores e mais ousados. h1 é o maior e mais ousado e h6 é o menor e menos ousado.
Qual tag HTML cria o menor cabeçalho?
Resposta: <h6>


Um elemento é o que chamamos de tag de abertura, tag de fechamento e código intermediário.
Qual destes é um elemento ?

<button>
</p>
<button>Hello, World!</button>
<h1>

Resposta: <button>Hello, World!</button>


Para criar uma página da Web, você precisará de uma tag HTML de abertura e fechamento.
Qual tag cria uma página HTML vazia ?
Resposta: <html>


Os elementos html e body formam a estrutura de uma página da web. Colocamos os elementos que queremos exibir dentro do corpo. Mais tarde, aprenderemos sobre outros elementos que entram <html> </html> para dar estilo às páginas da Web e torná-las interativas.


Qual tag define o corpo de uma página da web?
Resposta: <body>


Devemos colocar conteúdos, como cabeçalhos, dentro das tags do corpo.
